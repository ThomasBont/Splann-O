# Splanno

**Chat-first planning with shared expenses.**

Splanno is a mobile-first web application designed to help groups plan trips and events while managing shared expenses in one place.

Instead of switching between chat apps and expense splitters, Splanno combines:

- group chat
- shared expenses
- settlements
- planning tools

into a single experience.

---

## Features

- Real-time group chat
- Shared expenses
- Automatic settlement suggestions
- Settle-now payments
- Mobile-first UI
- Push notifications
- Progressive Web App (PWA)
- Invite friends to plans

Designed for:

- trips
- parties
- festivals
- group events

## Project Goal

Planning trips or group events usually requires multiple tools:

- messaging apps for communication
- expense splitting apps
- planning documents or notes

Splanno explores a **chat-first approach to group coordination**, where planning and expenses happen in the same place.

The project focuses on combining:

- real-time communication
- collaborative planning
- shared financial tracking

in a single mobile-first experience.
